# 公开接口

## 获取交易所当前所有交易对

此 API 用于获取交易所当前所有交易对。

### HTTP Request

`GET /v2/public/contracts/symbols`

## 获取时间
此 API 用于获取系统时间

### HTTP Request
`GET /v2/public/server-time`

## 获取币种

此 API 用于获取币种

### HTTP Request
`GET /v2/public/contracts/currencies`

## 获取指数列表

此 API 用于获取指数列表

### HTTP Request
`GET /v2/public/indexes/symbols`
